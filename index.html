Text Twist

View project on GitHub

Feature

  • A word puzzle web application written in Javascript
  • With session and cookie function, users can automatically login in and resum the game
  • All answer is given by a sqlite database on server side
  • Players have multiple choices like twist the given letter set and show whole answer

Detail

Text Twist is a simple word puzzle. Players first are given a random set of characters and need to guess the words composed by these characters. The answer set is fixed and players have multiple choices like twist the given characters, show all answer and restart the game.


The whole workspace contains serveral php files, one index page and one sqlite database. In the index page, functions are written in Javascript. One php file generates the random set of characters and the other is responsible for database query. The index page communicate with php files by AJAX and all the data is sent back in JSON format. With the php session, this game can remember previous user's name and game record, whhich is exactly the cookie function.
