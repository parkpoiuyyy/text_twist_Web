Gee-mail

View project on GitHub

Feature

  • Web based user interface
  • Secure mail service system supported by database server
  • All data stored in hash style except for username
  • Required passphrase to encrypt and decrypt message by salsa20

Detail

Gee-mail is a mail service system with security consideration, including login,register, read message and write message function. The source code is written in C++ and the executable file is called in php.


We create the database in SQLite3. There are two tables. Table user contains user and passwd. Table message contains column message id, send, receiver, cipher message. All stored data is encrypted except for username.


In our design, it requires users to type in the passphrase to read or write message. Passphrase is the secret between every two users and we are not allowed to store it in database.
