Text Twist

View project on GitHub

Feature

  • A word puzzle web application written in Javascript and PHP
  • All answer is given by a sqlite database
  • Players have multiple choice like twist the given letter set and show whole answer

Detail

Text Twist is a simple word puzzle. Players first are given a random set of characters and need to guess the words composed by this characters. The answer set is fixed and players have multiple choices like twist the given characters, show all answers and restart.


The whole workspace contains two php files, one index page and one sqlite database. In the index page, several functions are written in Javascript. One php file generates the random set of characters and the other is responsible for database query. The index page communicate with php files by AJAX and all the data is sent back in JSON format.
